# 文献检索下载工作流 Skill

## 概述
从零开始检索、下载、整理文献并导入Zotero的完整流程。适用于中文（CNKI）和英文（IEEE/Elsevier/Springer/Wiley/MDPI）论文的系统性收集。

## 适用场景
- 系统性文献综述
- 研究课题文献收集
- 多数据库交叉检索

---

## 一、检索阶段

### 1.1 Google Scholar / Web 检索（发现论文）

**成功的做法：**
- 使用 WebSearch 工具进行关键词检索（避免 Google Scholar 网页版的 CAPTCHA）
- 每次搜索使用 2-3 个精确关键词组合，而非泛泛搜索
- 搜索结果记录：论文标题、期刊名、年份、DOI/链接
- 区分论文来源数据库（IEEE / Elsevier / Springer / MDPI / Wiley）

**避免的做法：**
- ❌ 直接访问 scholar.google.com — 会被 CAPTCHA 拦截
- ❌ 使用过于宽泛的关键词 — 结果太多且不精确
- ❌ 同时搜索多个不相关方向 — 难以管理结果

### 1.2 图书馆统一检索（定位全文）

**流程：**
1. 打开图书馆首页
2. 在"搜索纸本+电子的全文资源"输入框粘贴论文标题
3. 点击搜索
4. 在结果中找到"馆藏电子资源"区域
5. 点击对应的数据库链接（IEEE / ScienceDirect / Springer 等）

**成功的做法：**
- 用完整论文标题搜索，图书馆统一检索比单个数据库更全面
- "馆藏电子资源"标签下的链接经过代理认证，可直接跳转

## 二、下载阶段

### 2.1 IEEE Xplore（推荐优先使用）✅

**使用 `ieee-download` skill 自动化下载。** 详见 `ieee.skill/ieee-download/SKILL.md`

**自动化下载流程：**
1. 导航到 IEEE 论文页（通过图书馆代理）
2. `evaluate_script` 提取 `arnumber`
3. 直接导航到：`https://ieeexplore.ieee.org/stampPDF/getPDF.jsp?tp=&arnumber=NUMBER`
4. 无超时 → PDF 自动下载；超时 → 告知用户 `Ctrl+S` 手动保存
5. 下载后自动复制到 `<workspace>/ziliao/`，确认后删除源文件

**成功的做法：**
- IEEE 通过图书馆 IP 认证，无需额外登录
- 小文件（<2MB）30秒内下载完成
- 知道 arnumber 时可跳过论文页直接下载

### 2.2 Springer ✅

**使用 `springer-download` skill 自动化下载。** 详见 `springer.skill/springer-download/SKILL.md`

**流程：**
1. 导航到 Springer 论文页（通过图书馆代理）
2. 点击 "Download PDF" 或直接导航到 `https://link.springer.com/content/pdf/DOI.pdf`
3. PDF 自动下载
4. 下载后自动复制到 `<workspace>/ziliao/`，确认后删除源文件

### 2.3 Elsevier ScienceDirect ⚠️

**问题：**
- 图书馆代理跳转到 ScienceDirect 后触发 "Are you a robot?" 检测
- IP 不在校园网范围时搜索 API 直接封锁
- 自动化脚本无法通过 reCAPTCHA

**替代方案：**
- ❌ 不要尝试用 curl/脚本直接访问 ScienceDirect
- ✅ 手动在浏览器中完成下载
- ✅ 通过图书馆 VPN 登录后手动访问

### 2.4 MDPI（开放获取）⚠️

**问题：**
- curl 下载返回 403 Access Denied
- MDPI 使用 Akamai CDN 阻止非浏览器请求

**替代方案：**
- 浏览器中打开 MDPI 文章页 → 点击 Download PDF
- 通过图书馆统一检索也可定位到 MDPI 论文

### 2.5 CNKI 中文文献 ✅

**流程：**
1. 通过图书馆首页 → CNKI 链接进入（自动 IP 认证）
2. URL 直接搜索格式：`https://kns.cnki.net/kns8s/search?classid=YSTT4HG0&kw=关键词1+关键词2`
3. 导航到论文详情页（用 `navigate_page`，不要点击链接）
4. `wait_for` "PDF下载" → `click` 点击下载按钮
5. 文件自动保存到浏览器默认下载目录

**下载后处理（cnki-download skill 自动执行）：**
6. 检测浏览器默认下载目录（先查 `$USERPROFILE/Downloads/`，若无则询问用户）
7. 逐文件复制 PDF 到工作目录 `<workspace>/ziliao/`（不用通配符，避免中文文件名问题）
8. 向用户确认后逐文件删除源文件

**注意事项：**
- 下载操作默认有权限，直接点击 PDF 下载即可
- 出现拼图验证码时需手动完成（检查 `#tcaptcha_transform_dy` 是否可见）
- 期刊官网（如航空学报 hkxb.buaa.edu.cn）直接下载无需验证码
- 批量下载时：先全部下载 → 统一复制到工作目录 → 确认后统一删除源文件
- **文件操作安全**: 逐文件用明确文件名操作，不依赖通配符处理中文路径

### 2.6 Google Scholar 网页搜索 ⚠️

**问题：**
- 频繁查询后触发 Google "unusual traffic" 检测
- 页面显示 reCAPTCHA 或 "sorry/index" 页面
- 同一 IP 多次查询后完全封锁

**替代方案：**
- ✅ 使用 WebSearch 工具进行学术搜索
- ✅ 搜索时添加 `site:ieeexplore.ieee.org` 等域名过滤

### 2.7 Web of Science ⚠️

**问题：**
- 图书馆代理跳转返回 403
- 直接访问需要个人账号登录
- IP 认证不适用于 WOS

**替代方案：**
- 使用 IEEE Xplore 和 Google Scholar 作为替代检索工具
- 或通过学校 VPN 客户端连接后访问

### 2.8 未覆盖数据库：自动创建 Skill 🔧

遇到尚无下载 skill 的数据库时（如 Wiley、Taylor & Francis、SAGE、ACM 等），**不要直接手动下载**，而是：

1. **分析页面结构**：导航到论文详情页，用 `take_snapshot` + `evaluate_script` 分析：
   - PDF下载按钮的 selector
   - 页面加载特征
   - 是否需要登录/验证码
   - PDF直链格式（如 `content/pdf/DOI`）

2. **尝试下载**：用分析出的方法点击PDF下载，确认能触发下载

3. **撰写 Skill**：按 `ieee-download/SKILL.md` 的模板格式，创建 `<数据库名>-download/SKILL.md`：
   - 命名：`<数据库名>.skill/<数据库名>-download/SKILL.md`
   - 写入导航→下载→后处理完整流程
   - 记录验证过的 selector 和注意事项
   - 放入对应 `.skill/` 文件夹

4. **更新工作流**：在本文件的数据库对比表中添加新条目

**模板结构**：
```markdown
---
name: xxx-download
description: Download paper PDF from XXX via library proxy.
argument-hint: "[paper URL or DOI]"
---

# XXX Paper Download

## Prerequisites
- 通过学校图书馆代理访问（自动IP认证）

## Steps
### 1. Navigate to paper page
### 2. Click/download PDF
### 3. Post-download（复制到ziliao + 确认删除源文件）

## Direct download shortcut（如有直链格式）
## Verified selectors
```

这样技能库自动生长，下次遇到同数据库论文就能直接用。

## 三、文件整理阶段

### 3.1 命名规范

**英文论文：** `中科院分区_数据库_期刊缩写_论文简称.pdf`

| 元素 | 说明 | 示例 |
|------|------|------|
| 分区 | Q1 / Q2 / NA（待查） | Q1 |
| 数据库 | IEEE / Elsevier / MDPI / Springer / Wiley | IEEE |
| 期刊缩写 | TMC / TITS / IoT-J / CJA / CN | IoT-J |
| 论文简称 | 精简英文关键词 | QoS_Multi_UAV_Offloading |

**中文论文：** `论文标题_第一作者.pdf`

### 3.2 重点论文标记

与研究方向最契合的核心论文加 `重点_` 前缀，便于优先阅读：
```
重点_Q1_IEEE_JournalAbbr_ShortTitle.pdf
```

### 3.3 文件夹分类

按研究主题创建子目录，不相关论文放入 `_backup/`：
```
papers/
├── 主题一_XXX/
├── 主题二_XXX/
├── 主题三_XXX/
├── 综述/
└── _backup/

## 四、Zotero 导入阶段

### 4.1 导入方式

**HTTP MCP 端点（curl + 文件方式）：**

```bash
# 1. 创建 item
curl -s -X POST "http://127.0.0.1:23120/mcp" \
  -H "Content-Type: application/json; charset=utf-8" \
  -d @payload.json -o response.json

# 2. 从 response 提取 itemKey
# 3. 添加到收藏集
curl -s -X POST "http://127.0.0.1:23120/mcp" \
  -H "Content-Type: application/json; charset=utf-8" \
  -d @add_to_collection.json -o response.json
```

### 4.2 关键经验

**成功的做法：**
- ✅ 用 Python 生成 JSON 请求文件，通过 `curl -d @file` 发送
- ✅ 文件编码使用 UTF-8
- ✅ 收藏集先用英文名创建，再用中文名重建后移动条目
- ✅ `Content-Type` 头加 `charset=utf-8`

**避免的做法：**
- ❌ 不要用 `curl -d '{...inline...}'` 传中文 — 乱码
- ❌ 不要用 Python `requests` 库调 MCP — 返回 502
- ❌ 不要用 Python `print` 打印中文到 stdout — cp1252 崩溃
- ❌ Zotero MCP 工具（`mcp__zotero__*`）经常返回 403，优先用 curl HTTP 端点

### 4.3 收藏集命名

**先创建英文收藏集，再迁移到中文（避免编码问题）：**

```
Step 1: 创建英文临时收藏集 → 导入论文
Step 2: 用中文名创建正式收藏集
Step 3: add_items_to_collection 将论文从英文集移到中文集
Step 4: 手动删除英文临时收藏集
```

### 4.4 批量导入脚本模板

```python
import json, subprocess, os

mcp = 'http://127.0.0.1:23120/mcp'
items = [...]  # 论文列表

for item in items:
    # Create item
    payload = json.dumps({
        'jsonrpc': '2.0', 'id': '1',
        'method': 'tools/call',
        'params': {
            'name': 'write_item',
            'arguments': {
                'action': 'create',
                'itemType': 'journalArticle',
                'fields': {'title': title, 'date': year, 'publicationTitle': journal}
            }
        }
    }, ensure_ascii=False)
    
    # Write to file and send
    with open('payload.json', 'w', encoding='utf-8') as f:
        f.write(payload)
    
    subprocess.run(['curl', '-s', '-X', 'POST', mcp,
                   '-H', 'Content-Type: application/json; charset=utf-8',
                   '-d', '@payload.json', '-o', 'response.json'], timeout=15)
    
    # Parse response and extract itemKey
    with open('response.json', 'r', encoding='utf-8') as f:
        data = json.load(f)
    item_key = json.loads(data['result']['content'][0]['text'])['data']['itemKey']
    
    # Add to collection
    add_payload = json.dumps({
        'jsonrpc': '2.0', 'id': '2',
        'method': 'tools/call',
        'params': {
            'name': 'add_items_to_collection',
            'arguments': {'collectionKey': collection_key, 'itemKeys': [item_key]}
        }
    }, ensure_ascii=False)
    
    with open('add.json', 'w', encoding='utf-8') as f:
        f.write(add_payload)
    
    subprocess.run(['curl', '-s', '-X', 'POST', mcp,
                   '-H', 'Content-Type: application/json; charset=utf-8',
                   '-d', '@add.json', '-o', 'add_resp.json'], timeout=10)
```

## 五、各数据库对比总结

| 数据库 | 自动化程度 | 下载方式 | 注意 |
|--------|-----------|---------|------|
| IEEE Xplore | ⭐⭐⭐⭐⭐ | getPDF.jsp 直链 | 最优选择 |
| Springer | ⭐⭐⭐⭐ | content/pdf/DOI | 直链可用 |
| CNKI | ⭐⭐⭐⭐ | 浏览器点击下载 | 需手动验证码 |
| MDPI | ⭐⭐⭐ | 浏览器下载 | curl 返回 403 |
| Elsevier SD | ⭐⭐ | 需手动下载 | 机器人检测拦截 |
| Google Scholar | ⭐ | WebSearch 代替 | CAPTCHA 封锁 |
| WOS | ⭐ | VPN 登录 | 代理失效 |
| 其他数据库 | 🆕 自动创建 | 按 §2.8 流程 | 遇到新库自动写skill |

## 六、环境约束速查

| 场景 | 问题 | 解决方案 |
|------|------|---------|
| Python print 中文 | cp1252 崩溃 | 写入文件 |
| Python requests MCP | 502 | 用 curl + subprocess |
| curl 内联中文 | 乱码 | `-d @file.json` + UTF-8 |
| 文件路径含中文 | bash 失败 | 用 `C:/temp_zotero/` |
| CNKI 验证码 | 搜索结果为空 | 手动完成拼图 |
| Google Scholar | CAPTCHA | 用 WebSearch 工具 |
| Zotero MCP 工具 | 403 Forbidden | 用 curl HTTP 端点 |
| ScienceDirect | robot detection | 手动登录VPN后下载 |
